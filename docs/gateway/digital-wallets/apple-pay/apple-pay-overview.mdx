The Apple Pay guides provide comprehensive instructions for integrating Apple Pay payments using Fondy, enabling seamless and secure transactions for both web and mobile platforms. The following guides outline multiple integration options tailored to your requirements and technical stack.

## Web Integration Options

* [Embedded Apple Pay Button Integration](/gateway/digital-wallets/apple-pay/apple-pay-web-integration#embedded-apple-pay-button-integration): Enhance the user experience by embedding the Apple Pay button directly into your website.
* [Direct Integration with Apple Pay API](/gateway/digital-wallets/apple-pay/apple-pay-web-integration#direct-integration-with-apple-pay-api): Gain full control over the payment flow by integrating directly with the Apple Pay API.

## Mobile Integration

Apple Pay can be implemented in your [mobile application](/gateway/digital-wallets/apple-pay/apple-pay-mobile-integration) using the Fondy SDK, with support for the following programming languages and frameworks:

* Objective-C
* Swift
* React Native
* Flutter
