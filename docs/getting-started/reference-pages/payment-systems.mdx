The payment methods supported by Fondy are categorized into payment systems. These payment system categories are used when selecting the payment types available to your customers when using Fondy Solutions. The following table lists and describes the existing payment system codes you can use when setting a payment with one of the Fondy solutions:

| Payment system code | Description                   |
| ------------------- | ----------------------------- |
| card                | Visa/MasterCard/Maestro       |
| banklinks\_pl       | Polland bank links            |
| banklinks\_eu       | European country's bank links |
| installments        | Installments                  |
